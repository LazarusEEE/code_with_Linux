# Linux Projects
A collection of Linux-based projects, system utilities, shell scripts, and automation tools developed to improve my understanding of Linux operating systems and system programming.

## About Me
B.Sc. Electrical and Electronic Engineering (EEE) student.  
This repository contains my Linux projects created while learning operating systems, command-line tools, scripting, and system management.

## Projects
### 1. Linux System Monitor
A command-line tool to monitor system resources such as CPU usage, memory usage, disk space, and running processes.

**Technologies:**
- Bash Script
- Linux Commands

**Features:**
- Display CPU information
- Monitor RAM usage
- Check disk space
- View active processes


### 2. Automated Backup Script
A shell script that automatically creates backups of important files and directories.
**Technologies:**
- Bash
- Cron Jobs

**Features:**
- Automatic backup creation
- Date-based backup naming
- File compression


### 3. File Management Utility
A Linux command-line application for managing files and directories.

**Features:**
- Create files and folders
- Search files
- Move and delete files
- Display file information
